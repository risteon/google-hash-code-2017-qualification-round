============
Installation
============


Stable release
--------------

To install hashcode-2017-qualification, run this command in your terminal:

.. code-block:: console

    $ pip install hashcode_2017_qualification

This is the preferred method to install hashcode-2017-qualification, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for hashcode-2017-qualification can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/risteon/hashcode_2017_qualification

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/risteon/hashcode_2017_qualification/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/risteon/hashcode_2017_qualification
.. _tarball: https://github.com/risteon/hashcode_2017_qualification/tarball/master
